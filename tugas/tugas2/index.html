Tugas 2: JavaScript Interaktif

Membuat aplikasi interaktif dengan JavaScript vanilla

Deskripsi Tugas

Tugas ini bertujuan untuk mengembangkan pemahaman siswa tentang JavaScript dan bagaimana menggunakannya untuk membuat interaksi yang dinamis di halaman web. Siswa harus membuat aplikasi atau fitur interaktif yang merespons input pengguna dengan baik.

Kriteria Tugas:

  • Menggunakan JavaScript vanilla (tanpa framework)
  • Membuat minimal 3 interaksi/event handler
  • Manipulasi DOM yang efektif dan efisien
  • Menggunakan variabel dan function dengan baik
  • Kode yang bersih dan mudah dipahami
  • User experience yang intuitif
