Tugas 6: API Integration

Mengintegrasikan aplikasi dengan API eksternal

Deskripsi Tugas

Tugas ini mengajarkan siswa untuk bekerja dengan API eksternal, yang merupakan skill penting dalam pengembangan web modern. Siswa harus mampu melakukan request ke API, menangani response, dan menampilkan data dengan cara yang user-friendly.

Kriteria Tugas:

  • Mengintegrasikan minimal 1 public API
  • Menggunakan fetch API atau axios untuk HTTP request
  • Error handling yang baik untuk API failures
  • Loading state dan user feedback yang jelas
  • Caching atau optimization untuk reduce API calls
  • Display data dengan format yang menarik
  • Dokumentasi tentang API yang digunakan
