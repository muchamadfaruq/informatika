Tugas 1: Web Design

Membuat halaman web dasar dengan HTML dan CSS

Deskripsi Tugas

Dalam tugas ini, siswa diminta untuk membuat halaman web dasar menggunakan HTML dan CSS. Tujuan utama adalah memahami struktur dokumen HTML dan bagaimana styling CSS dapat mengubah tampilan visual sebuah website.

Kriteria Tugas:

  • Membuat struktur HTML yang valid dan semantis
  • Menggunakan CSS untuk styling yang menarik
  • Responsif pada berbagai ukuran layar (mobile, tablet, desktop)
  • Menggunakan minimal 5 elemen HTML berbeda
  • Implementasi layout yang rapi dan terorganisir
